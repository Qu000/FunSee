# FunSee
是之前FunSee_iOS的延续版，FunSee_iOS存在一些架构问题，在此项目库中修正。

## tabBar含4个item   
home、Fun看、艺美、乐我

## home   

搞笑段子、趣味图片、上拉刷新、下拉刷新   

## Fun看   

爆笑视频、自定义播放器、tableView ---- cell播放、跳转页面播放  

## 艺美   

艺术散文、美文、诗歌，艺术视频，圆点push、pop转场动画  

## 乐我   

支持两种排列方式：悬浮、collection；  ARKit小游戏；  清除缓存；  联系我们；  搜索

###### 搜索   

支持语音搜索----原生框架
